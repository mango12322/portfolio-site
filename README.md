##프로젝트 개요
웹 퍼블리셔 취업을 목표로 제작한 개인 포트폴리오 웹사이트입니다.
사용자의 시선 흐름을 고려한 UI 설계와 반응형 레이아웃을 통해 다양한 디바이스 환경에서도 최적의 사용자 경험을 제공하도록 구현했습니다.
자기소개, 기술 스택, 프로젝트 소개 등으로 구성하여 개인의 역량을 효과적으로 전달하는 데 중점을 두었습니다.

##사용 기술
HTML5 – 시맨틱 마크업을 활용한 구조 설계
CSS3 – 스타일링 및 유지보수를 고려한 구조화
JavaScript (Vanilla JS) – 인터랙션 및 동적 기능 구현
Responsive Web – 미디어 쿼리를 활용한 반응형 디자인
Git / GitHub – 버전 관리 및 프로젝트 관리 

##주요 구현 기능
반응형 웹 구현 (PC / Tablet / Mobile 대응)
스크롤 기반 애니메이션 및 인터랙션 효과
기술 스택 및 프로젝트를 시각적으로 표현한 UI 구성
사용자 경험을 고려한 네비게이션 및 레이아웃 설계
크로스 브라우징을 고려한 스타일 및 기능 구현
